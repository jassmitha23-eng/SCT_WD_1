# 🌐 Responsive Landing Page

This project is a modern and responsive landing page developed as part of my internship task. It demonstrates the use of HTML, CSS, and JavaScript to build an interactive and visually appealing web interface.

---

## 🚀 Features

- 📌 Fixed Navigation Bar
- 🎯 Navbar changes color on scroll
- ✨ Smooth scrolling navigation
- 🔘 Interactive "Explore More" button
- 🎨 Modern UI with gradient design
- 🖼️ Image-based service cards
- 📱 Fully responsive layout

- CSS3  
- JavaScript
